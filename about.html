About GeoJson

GeoJson is part of Javascript Object Notation, a lightweight data interchange format that is easy to read and write for programmers. GeoJson is used for encrypting a variety of geographical data structures. It is usually used as an interface for applications to represent features or collection of features if needed. It describes several types of JSON Objects and the way in which they are combined to represent a geospatial data.

It supports a variety of geometry types such as Point, LineString, Polygon, MultiPoint, MultiLineString and MultiPolygon.
